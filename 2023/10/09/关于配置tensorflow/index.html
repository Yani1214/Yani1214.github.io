关于配置tensorflow

发表于 2023-10-09 更新于 2023-10-26 分类于 大数据分析及隐私保护 阅读次数：

在配置tensorflow1.9.0时，遇到了一些问题，遂进行记录

基本步骤

  • anaconda建立新环境
  • 在新环境中通过conda install tensorflow=1.9.0进行下载，从而避免了依赖包的问题

遇到的问题

  • ImportError：DLL load failed:找不到模块
  • 解决方法见网址：解决ImportError：DLL load failed:找不到模块，Failed to load the native TensorFlow runtime._importerror: dll load failed: 找不到指定的模块。 failed to -CSDN博客

总结

  • pip install有时候不会把依赖包下完，需要conda install进行补充
  • python3.5不支持tensorflow1.9.0，需要改为python3.6
