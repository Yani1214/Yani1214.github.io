Web之CSS(1)——继承、层叠、特指、布局

拓展：伪类和伪元素

可以根据悬停时的状态对元素状态进行改变

【标签】:伪类名称 {}

【标签】::伪元素 {}

CSS

继承

  • 大部分属性都可继承，只有少部分不能继承
  • 优先级很低，低于通配符选择器*，因此会被覆盖

层叠

五个来源

  • 浏览器默认CSS
  • 用户CSS：用户可以自行放大缩小页面
  • 网页作者链接CSS
  • 网页作者内嵌CSS
  • 网页作者行内CSS（优先级最高）

确定步骤

  • 收集某个标签目前拥有的所有样式：进行遍历和罗列
  • 根据优先级和特指度选择样式
  • 根据样式定义位置进行选择：最后定义的样式会覆盖之前先定义的样式

PS: 如果某个样式存在!import，则表示为最重要的样式

特指度

  • I-C-E：伪类为C，伪元素为E
  • I 100 C 10 E 1，不存在进位；即只要有I，C再大没有I也比有I的小

布局

盒子模型

  • 任何网页都是盒子

组成（从内到外）

  • content：盒子的内容
  • padding：增加content和border之间的空白
    • 一共有四个方向 起始方向是上 顺时针推导
    • 一个值 设定的是四边
    • 两个值 设定的是上和右 下对应上 左对应右
    • 三个值 设定的是上 右 下 左对应右
    • 四个值 设定的是上 右 下 左
  • border：元素外边框，默认设置为不可见
  • margin：确定每个元素和其他元素之间的间隔

两个基本属性

  • height：高度，不用设置，会自动伸缩
  • width：宽度，浏览器宽度有限，一般不设置滚动条，需要仔细计算
    • 非IE 只包含了content宽度
    • IE 包含了content和padding宽度

宽度设置

  • 元素没有设置width
    • 会继承父类的width，content会被压缩
    • 要么为border宽度
  • 元素设置了width，则content会向外扩张，有可能会转行
  • 两个元素左右间隔=左边margin+右边margin
  • 两个元素上下间隔=max{上边margin，下边margin}（高度不叠加）
  • content padding border margin之和为一个元素的宽度

浮动

特点

  • float，包括left和right两个值
  • 会从标准文档流中脱离，为最上面的图层；即浮动的遮挡标准的
  • 浮动的元素的display会自动变为inline-block
    • 宽度由元素内容决定
    • 可并栏显示
  • 会移动到浮动元素的旁边进行等高排列 或 移动到非浮动元素所在行的最左边/最右边（覆盖）
  • 不会遮挡文字

作用

  • 实现图文绕排

    • 图片在上方 文字在下方

    • 图片设置为左浮动

  • 实现块状元素的并栏显示

    • 均设置为左浮动

带来的问题及其解决方法

  • 高度坍塌：父元素没有元素了，变成黑线；子元素为浮动元素
    • 父元素也设为浮动
    • overflow设置为hidden，不允许其中元素超出设定的高度，盒子外面的内容不显示；即不允许改变父元素高度，此时父元素仍然为标准文档流
    • 在下面再增加一个不可浮动的盒子，并设置清除浮动clear：left right both，指定左边和右边不能有浮动元素；这样只能使得浮动元素出现在上方
  • inline-block属性不要和float连用

显示

display

  • inline：元素从左到右排列，不能容纳时会自动换行，不可设置高度
  • inline-block：
    • 元素从左到右排列
    • 元素可以设置高度
    • 元素宽度由元素内容决定，不由设定的width决定
  • block：元素独占一行，可以设置宽度和高度
  • flex
  • grid
  • none：不展示该元素，且该元素原先的位置不保留

定位

  • position表示

四个属性

  • top
  • left
  • right
  • bottom

三种定位

  • 相对定位 relative
  • 固定定位 fixed
  • 绝对定位 absolute
    • 以body元素作为定位进行偏移
    • 如果父元素有设置position，则将其作为定位进行偏移
