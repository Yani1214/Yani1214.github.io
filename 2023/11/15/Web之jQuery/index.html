Web之jQuery

jQuery概念

定义

$(selector).action()

  • $定义了jQuery，为基本单位
  • selector定义了HTML元素
  • action()对selector元素进行相应操作
  • 整体定义了一个完整的事件

语法

  • $(document).ready(function)(){}：初始化函数，表示网页的DOM数已经加载好了，然后执行接下来的操作，与$(function()){}含义相同

jQuery选择器

本质

  • CSS选择器

类型

  1. 基本选择器
  • 元素选择器
  • ID选择器 #
  • 类选择器 .
  1. 层次选择器
  • 父子选择器 >
  • 祖先选择器 grand
  • 临近兄弟选择器 +
  • 兄弟选择器 ~
  1. 属性选择器
  • name |= value 所有以value为前缀的属性（value-）
  • name ~= value 所有包含了value的属性（[空格]value）
