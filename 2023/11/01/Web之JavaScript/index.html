Web之JavaScript

JS概念

作用

  • 实现网络用户交互
  • 进行DOM（对网页内容进行动态增删改查）和BOM（对浏览器的动态操作）操作
    • BOM和DOM为API，不同浏览器API不同，从而导致二者没有标准；所以存在一个JQuery库（存储接口函数），只要调用该库，程序在任何浏览器上均可以跑通
    • js语法是ECMAScript定义的标准
  • Atom用于js开发，可从网页开发扩展至桌面开发

运行方式

  • node + 程序名.js

  • 嵌入至浏览器中运行，运用<Script>标签；但js实际开发时应该放在<body>的结束符上方

    • 可以加快网页加载速度：加载脚本需要时间，所以加载速度会延迟
    • 解决网页内容加载问题：脚本先加载了，但网页内容还没加载，因此会出现错误

  • 通过src链接到外部的js程序运行（要单独创建一个同源目录，可以实现样式和代码分离）

  • F12打开网页控制台进行js代码编写，用于对网页进行调试与验证，也可以用于做js实验（如正则表达式提取等）

JS语法

数据表示

  • 直接数据：直接看到的数据，如3.14、“GOOD”等
  • 变量：表示内存中某块存储区域的名字；变量赋值采用基于引用的方法（即某个存储空间存了某个字符串，变量若等于该值，相当于把该变量名的“标签”贴到该存储空间处；一个存储空间可以贴多个标签）
    • “望名知其意”
    • 匈牙利命名法：第一个单词全小写，从第二个单词后的每个首字母都进行大写表示

特点

  • 弱类型语言：定义了变量后，不限制其类型（c语言为强类型语言，即指定变量类型后，整个生命周期中该变量类型不可变动）
  • 变量大小写不一样
  • 变量名通过var进行定义（一般定义局部变量）

JS数据类型

基本数据类型

  • 字符串型String：用””包围（一般输入时均为字符串形式）
    • toString()：所有类型都默认携带的方法，将其转换为字符串型
    • Number()：强制将字符串转换为数值（只能包含数字字符）
    • parseInt()：转为整型
    • parseFloat()：转为浮点型
  • 数值型Number：整型和浮点数的集合
  • 布尔型Bool：true和false，全部小写

复合数据类型

  • 数组：每个元素可为不同类型
  • 对象：数据类型既有属性，又有方法（面向原型）

其他

  • Null：未对变量进行定义
  • Undefined：定义了变量，但是没有初始化
  • NaN：not a number，数值运算时无法得到一个数值结果
  • Infinity：数值计算时得到一个无限大的值
