从0开始的游戏开发生活（零）——游戏设计师攻略

序言

思来想去，最终还是下定决心学习游戏制作。可以说，游戏设计师是集成了本鼠鼠所有梦想的工作（可以同时满足电影、动漫、游戏、音乐、写作欲）。因为自己对游戏很感兴趣（4岁就开始玩游戏了），所以从中学到大学，我断断续续一直都有当游戏设计师的想法，但每次都会被现实击溃——每当看了国内游戏的行情现状后，会一次又一次地产生动摇，觉得浑浑噩噩一辈子也挺好（大学三年没学到什么技术 + 悲观主义）。

转折应该是发生在寒假玩了p5r之后。放寒假之前的冬促看到了p5r打折，当时对p5r的独特美术风格早有耳闻，但是不知道自己喜不喜欢jrpg的玩法，结果就是真香，每天不玩浑身难受（宫殿真好玩）。然后刷b站的时候又看到了p3r的发售，不得不感叹阿特拉斯做了对饭，美术音乐玩法是一点没有落下，理哥也是真帅。而因为p5r和p3r，让我对整个p系列产生了兴趣，于是我把剩下的4、2、1都粗略了解了一下，后面也开始看2和1的实况（4本鼠鼠要自己玩）。之后由p系列顺藤摸瓜，我开始了解阿特拉斯的游戏制作历史和正传真女神转生的相关内容。

答辩完后的一天下午，我盯着副岛画的壁纸，听着目黑大明神的音乐，感慨着自己和别人合作如果也能做出这些集大成作（美术、音乐、玩法、ui、剧情）的游戏那该是多么酷炫的一件事情。于是，我突发奇想，反正大三下了也很闲（基本没课了），还不如就尝试认真学习一个游戏引擎，不仅可以满足自己的快乐，也对以后找工作有帮助。自己最近也积累了一定的脑洞（本鼠鼠灵光一闪的时候都会记录自己的一些所谓的创意想法，为以后的艺术创作打基础），然后也有一定的计算机基础（苦逼网安人），音乐方面也开始有了感觉（最近很喜欢R&B和爵士），美术方面也一直都想好好学基础，不如现在就开始着手干起来，边做边学，这也比做安全更让我有动力。毕竟这个世界干什么都一样烂，还不如干自己能够坚持下去的。

于是，出于想要逼自己一把的原则，我开始搜集游戏开发相关的内容，并在知乎上找到了一篇关于独立游戏开发的帖子：https://www.zhihu.com/question/21031559 ，@窦月汐 答主非常细致的描述了游戏开发入门所需要的各种东西，包括思想、技术、设计等方面。在接下来的内容中我将对该帖子提到的重点内容进行归纳整理，有兴趣的uu也可以访问原帖支持答主。

最后就是，希望本鼠鼠可以坚持到底，如果真的可以做出一个有深度、有特色的游戏，并让玩家有所收获，我一定会非常开心，毕竟这才是游戏对于玩家和开发者的真正意义。祝我成功吧，EI Psy Kongroo。

游戏制作基本思想

设计理论——是什么、怎么做

  • 什么是好玩？——决定了会设计出怎样的游戏

    • 会玩，玩的足够丰富
    • 将感到”好玩“的经验进行整理
    • 最后进行设计理论构建

  • 设计理论构建

    • 游戏的定义是什么
    • 游戏的构成有哪些
    • 游戏的必须构成有哪些
    • 游戏的分类怎么分
    • 游戏的节奏怎么做
    • 游戏的反馈机制怎么做
    • 游戏的世界怎么构建
    • 判断游戏好坏的标准是什么
    • 游戏的终极目标是什么
    • 做游戏的目的是什么

  • 优秀的设计理论

    • 能够提供最基本的功能框架
    • 能够适用于多种对象
    • 指出了创作空间的范围

  • 参考资料

    • 游戏策划推荐书单：https://www.zhihu.com/tardis/zm/art/67963068?source_id=1003

    • 马伯庸的作家思维训练方法：

      https://www.zhihu.com/question/22460344/answer/22122699

创作理念——为什么、做什么

  • 分类
    • 为个体利益而创作
    • 为个人探索而创作
    • 为他人而创作
  • 核心
    • 三观决定了创作理念
    • 做游戏之前先树立好自己的三观
  • 体现
    • 游戏体验反映设计师创作理念
    • 需要对游戏构成要素有深刻理解

游戏引擎学习

引擎选择

  • Unity：适合2D小体量游戏（在这里选择Unity）
  • Unreal：适合3D大体量游戏

教程选择

  • 如何搜索

    • B站：[xxx] + [零基础教学]
    • Youtube：[xxx] + [beginner tutorial]

  • 如何选择

    • 有示范操作——是什么，怎么用
    • 有出错时的解决方案——有问题的地方会进行示范并进行解决

ChatGPT辅助教学

  • 建立语境
    • 交代背景：我是一个完全不懂游戏制作的人，而且我也没有编程知识，现在我第一次开始学习Unity，有很多不懂的概念和知识
    • 扮演身份：我希望你在后面的对话中扮演一个Unity教学导师，用新手能够理解的方式教我使用Unity，解答我的疑惑
    • 具体问题：现在我希望你能够试着理解我的情况和需求，并给我提供学习建议
    • 回答要求：后面我可能会无法用准确地术语提问，如果后面你对我的问题理解比较模糊，我希望你也能够向我提问，以便帮我（也是帮你）搞清楚我的需求
  • GPT解释概念
  • GPT解释如何操作

编程语言学习

  • 编程语言和游戏引擎的关系
    • 游戏引擎集合了游戏开发所需的工具、组件、环境
    • 编程语言负责将游戏引擎提供的内容进行组合
  • 学习路线
    • 先学编程语言
    • 当存在瓶颈后，转去学游戏引擎
    • 当存在瓶颈后，转去继续学编程语言
    • 对2、3步进行循环

游戏设计书籍阅读顺序

书单目标

  • 第一阶段：建立正确游戏设计师价值观
  • 第二阶段：拓展眼界，看见游戏更多可能
  • 第三阶段：系统认识电子游戏设计与开发基本流程，了解游戏设计师基本素养
  • 第四阶段：深入了解自己感兴趣的游戏类型，对专项问题进行研究

示例书单

  • 《快乐之道：游戏设计的黄金法则》——建立游戏设计道德基础

  • 《游戏改变世界》——进一步发现“游戏”更多可能性

    • 作者Jane的TED演讲：

      https://www.ted.com/talks/jane_mcgonigal_the_game_that_can_give_you_10_extra_years_of_life

  • 《全景探秘游戏设计艺术》——从“思想”到“设计”的过渡，游戏设计工艺流程的介绍

  • 《剑与电——角色扮演游戏设计艺术》——RPG游戏设计过程，对其他游戏研究有帮助

游戏美术学习

审美提高

  • 对游戏创作具有巨大作用的审美能力（Youtube搜索以下关键词即可）

  • 欣赏优秀作品
    • 多接触各种风格的游戏
    • 多逛设计师网站
      • ArtStation：https://www.artstation.com/
      • Pinterest：https://www.pinterest.com/
      • 花瓣网：https://huaban.com/

美术技法

  • 游戏制作过程了解
    • 阅读时间：约3h
    • 阅读资料：https://zhuanlan.zhihu.com/p/25292463
  • 美术技法分类
    • 审美能力：需要终生学习，持续提高
    • 绘画技术：2D绘画，传达设计样式
    • 建模技术：3D建模，制作相关模型（初学者建议从该步开始）
    • 动作技术：生物类模型的动作内容

建模工具

  • 游戏建模流程及相关工具

  • 核心工具——Blender
    • 官网：https://www.blender.org/
    • 中文社区：https://www.blendercn.org/
    • 学习目标
      • 快速过一遍Blender基础操作
      • 学习多边形建模：建一个机甲模型
      • 巩固多边形建模，掌握科学布线：建一个人体模型
      • 学习拆分UV和贴图绘制：给之前的机甲模型画贴图
      • 学习材质制作：给机甲模型做一套酷炫的材质
      • 学习雕刻：雕刻一个怪兽模型
      • 学习骨骼动画：为怪兽绑定骨骼，并摆几个pose

游戏创意评价

游戏评价标准

  • 乐趣评价——游戏好不好玩
    • 什么是好玩？
    • 为什么你认为一个游戏好玩，另外一个游戏不好玩？
    • 好玩的游戏有什么共同点？
  • 价值评价——游戏能给玩家带来什么
    • 满足一部分玩家的精神需求
    • 抚慰伤痛
  • 艺术评价——游戏在表达什么
    • 游戏可以是艺术媒介的一种
    • 制作者可以通过游戏去表达某种观点/某种情感/引起反思
